
VIX (Cboe Volatility Index) Datafile
=============================


DATASET DATES - 2005.Oct.03 -  2020.Jun.10 

File Format : {DateTime, Open, High, Low, Close}  
   - Times with zero volume are omitted (thus gaps in the data sequence are when there have been no trades) 
   - Excel will usually fail to open large files directly. To use the data in Excel, open the files using notepad and then break into smaller files or copy/paste into Excel
   
    NOTE : Exchanges sometimes includes out of hours data in the feed. These will often be sequences of identical   prices prior to or before the close. However, these datapoints will sometimes contain valid data that the exchange did not report in sequence during the official trading hours, as such they have not been removed.
